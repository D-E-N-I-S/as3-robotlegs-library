baseUrl:String

Set an optional base url as a prefix for all Assets.

If the requiest url of an Asset contains 'http://' the base url will not be added to the url.

load()method 
public function load(request:Object, closure:Function = null):IAsset

Load a single Asset.

The AssetLoader automatically assigns the right asset class by checking the extension of the asset. The following type's are currently supported:

  • BitmapAsset (png, gif, jpg and jpeg)
  • DisplayObjectAsset (swf)
  • SoundAsset (mp3, wav, aif)
  • CSSAsset (css)
  • XMLAsset (xml)

Parameters

request:Object — The url of the Asset as a String.
 
closure:Function (default = null) — Optional closure method for retrieving AssetLoaderEvents.

Returns
IAsset — A new instance of Asset.
